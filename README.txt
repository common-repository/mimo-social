This plugin creates a custom designed shares buttons in your posts. You hace shortcodes to show share buttons in posts and pages, or share counts of your profiles.

== Description ==

Create custom share bar in your posts or pages or create profile followers bar with shortcodes.

Use shortcode [mimo_social] to show sharing buttons.

Use shortcode [mimo_social_count] to show profile followers counts bar.

Go to Wordpress Dashboard/Settings/Mimo Social to set your social network API keys and data.

== Installation ==

1. Upload `mimo-social.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Wordpress Dashboard/Settings/Mimo Social to set your social network API keys and data
